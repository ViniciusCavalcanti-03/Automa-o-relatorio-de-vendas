Criei um script que simula uma rotina corporativa muito comum: a extração e o envio de relatórios diários de vendas.

O que o meu código faz na prática?
1️⃣ Abre o navegador e acessa o Google Drive corporativo.
2️⃣ Faz o download da base de dados de vendas do dia (Excel).
3️⃣ Utiliza a biblioteca Pandas para ler o arquivo e calcular indicadores cruciais: o faturamento total e a quantidade de produtos vendidos.
4️⃣ Acessa o Gmail, redige um e-mail formatado com esses resultados e realiza o envio automático para a diretoria/equipe.

🛠️ Tecnologias e Bibliotecas utilizadas:
* Python: Linguagem principal do projeto.
* Pandas: Para manipulação e análise dos dados da planilha.
* PyAutoGUI & Pyperclip: Para a automação da interface gráfica (mouse e teclado) e manipulação da área de transferência.
